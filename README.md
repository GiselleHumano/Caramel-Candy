# Caramel Candy

Proyecto final de React.js de CoderHouse.

Tienda e-commerce dedicada a la venta de dulces con tematica de Halloween.

# Tecnologías, librerías e íconos

• React Js
• Hooks utilizados: useState, useEffect y useContext
• React Toastify
• sweet alert 2
• React Icons
• Firebase
